Leetcode 136. Single Number

2020-04-16

This article offered two solutions on Leetcode 136, Single Number.

Problem description

Given a non-empty array of integers, every element appears twice except for one. Find that single one.
Requirement: Linear runtine complexity
Bonus: No extra memory

Solution

Solution 1- Set

Thoughts

We use a set to solve the problem. First, we iterate elements in the array. If the set contains the specific element, then remove it. Otherwise, add this element to the set. In this way, the set only contains the element which only appears once.

Code

import java.util.HashSet;

class Solution {
    
    public int singleNumber(int[] nums) {
        HashSet container = new HashSet ();
        for (int i=0; i < nums.length; i++){
            if(container.contains(nums[i])){
                container.remove(nums[i]);
            }else{
                container.add(nums[i]);
            }
        }
        
        Iterator value = container.iterator();
        int ans = -1;
        while(value.hasNext()){
            ans = (Integer)value.next();
        }
        return ans;
    }
    
}

Analysis

Run complexity: O(n)
Space complexity: O(n)
Cons: An ugly implementation when fetching the element from the HashSet

Solution 2-Magic of XOR

Thoughts

Some rules about xor may help:
a xor a = 0
a xor b = b xor a
a xor 0 = a

In this way, we can xor all of elements and the result will be the answer. For example, [5,1,2,1,2]1 xor 1 xor 2 xor 2 xor 5 = 0 xor 0 xor 5 = 5

Code

class Solution{
   public int singleNumber(int[] nums){
        for(int i=1; i< nums.length; i++){
            nums[0] = nums[i]^nums[0];       
        }
        return nums[0];
    }
}

Analysis

Time complexity: O(n)
No extra space
A efficient solution!
