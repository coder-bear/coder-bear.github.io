HashMap 与 Hashtable 区别

HashMap可以存放null, Hashtable不可以存放null HashMap是非线程安全类，Hashtable是线程安全类

StringBuffer和StringBuilder 区别

StringBuffer 是线程安全的 StringBuilder 是线程不安全的

单线程的StringBuilder操作比较快 多线程，为保证操作的正确性，需要使用StringBuffer

ArrayList 和 Vector的区别

ArrayList是非线程安全的，Vector是线程安全的。 使用 
Collections.synchronizedList
